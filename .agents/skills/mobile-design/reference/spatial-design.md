# Spatial Design (Phone-First)

- Design for narrow, tall canvases.
- Respect safe areas and gesture regions.
- Use consistent spacing rhythm (typically 4/8-based).
- Build hierarchy through spacing, alignment, and scale.
- Keep primary actions thumb-reachable.
- Mix compact and spacious regions intentionally to create rhythm.

Avoid:
- Uniform spacing everywhere (monotony)
- Top-heavy controls that are hard to reach
- Fixed pixel heights that break on smaller phones
