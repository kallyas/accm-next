# Typography (Flutter Mobile)

- Use clear hierarchy: display, title, body, label.
- Prefer platform-native defaults unless brand fonts are already established.
- Support large text scaling (up to ~200%) without clipped layouts.
- Keep body text comfortably readable (roughly 11sp/11pt minimum for dense UI, usually higher).
- Pair expressive headline typography with highly legible body typography.

Avoid:
- Flat, same-size text everywhere
- Decorative fonts for body copy
- Tight line height that hurts readability
