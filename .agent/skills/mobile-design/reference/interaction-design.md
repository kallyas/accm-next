# Interaction Design (Flutter Mobile)

- Minimum touch targets:
  - iOS: 44x44pt
  - Android: 48x48dp
- Preserve platform navigation and system gestures.
- Provide immediate visual/haptic feedback for taps and state changes.
- Prefer bottom sheets and contextual actions over disruptive modal overload.
- Build robust loading, error, success, and empty states.
- Use keyboard-appropriate input types and inline validation for forms.

Avoid:
- Tiny hit targets
- Hidden critical actions behind ambiguous icons
- Platform-inappropriate controls and gestures
